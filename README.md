# HDT3 - Algoritmos de Ordenamiento

Comparación de tiempos de ejecución de 5 algoritmos de sorting.

## Algoritmos
- Gnome Sort
- Merge Sort
- Quick Sort
- Radix Sort
- Heap Sort

## Compilar y Ejecutar

```bash
javac Main.java sorts/*.java
java Main
```

## Resultados

Se generan en `resultados/resultados.csv`
